# Logistic Regression and TF-IDF in Detecting Fake News

Data Source: 
<b> https://github.com/Tariq60/LIAR-PLUS/tree/master <b>

This poster visualizes the context, process, results, and ethics behind two crucial research questions:
1. What variables are associated with the validity of a political statement?
2. What linguistic features are associated with the binary validity of a political statement

This study was conducted with the following procedures below:
1. Multinomial Regression using SkLearn Package in Python
2. TF-IDF with TextBlob and other feature extractors

This breif posters contains the following information:
1. Data Manipulation Processes (pandas,numpy)
2. Preliminary data interpretation with visualizations (plotnine)
3. Training and Testing of Multinomial Logisitc Regression and TF-IDF machine learning models (sklearn, TextBlob)
4. An interpretation of standardized coefficients and model evaluation metrics such as accuracy and F1 score.
5. A discussion about the ethics of the data and results.

The data used in this study comes from the LIAR-PLUS dataset, built from fact-checking articles sourced from Politifact.com. Allegedly an independent, non-partisan, fact-checking website that evaluates the accuracy of claims made by U.S. politicians, political parties, and advocacy groups. It is run by journalists. Manually labeled data scraped from Politifact over a decade (2007 -2017) in timespan.
